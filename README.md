# b_users

**Description:**  
backend component handling domain/api/users requests


**API-rules**
```
Route                            | HTTP-Method        | Input-Information                                | Output-Information                                                 | Description                                     
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
/                                | POST               | username: String                                 | 200 - message: String
                                                      | email: String
                                                      | password: String                                 | token: String (added to the cookies)
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
/                                | DELETE             | token: String                                    | 200 - message: String                                              | extended with password for security/verification
                                                      | password: String                                 | (clears token from cookies)                                        
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
/create/google                   | POST               | /                                                | Not implemented
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
/create/apple                    | POST               | /                                                | Not implemented
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
/me                              | GET                | token: String                                    | 200 - username: String                                            | changed generalized notifications to lecturesSubscription
                                                                                                          | email: String                                                    | and subscribedAuthors for better access
                                                                                                          | permissionLevel: String
                                                                                                          | lecturesSubscription: boolean
                                                                                                          | subscribedAuthors: Json[userid: Integer, username: String]
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
/list                            | GET                | token: String                                     | 200 - users: Json[]
                                                      | (admin action)
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
/permission                      | PUT                | token: String                                     | 200 - message: String                                            | changed role to permissionLevel to fit the value type
                                                      | username: String
                                                      | permissionLevel: Integer (1,2 or 3)
                                                      | (admin action)
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
/password-change                 | PUT                | username: String                                  | 200 - message: String
                                                      | password: String
                                                      | newPassword: String (not identical to password)
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
/password-reset/:email           | GET                | /                                                | Not implemented
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
/password-reset/change/:token    | GET                | /                                                | Not implemented
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
/password-reset/change/:token    | GET                | /                                                | Not implemented
```
